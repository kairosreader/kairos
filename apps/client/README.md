# Client App

## Development

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

Select your platform:
- `i` - iOS
- `a` - Android
- `w` - web

## Structure

- `/app` - Screens and navigation
- `/assets` - Static assets
- `/components` - UI components
- `/constants` - Configuration

## Tech Stack

- Expo
- React Native
- Expo Router
